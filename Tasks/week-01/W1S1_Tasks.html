Exercise 1

Adding Some Headers

To add headers, we use the <h1></h1> tag to indicate the most important header. We can also use <h2></h2> through to <h6></h6>.

Adding some Paragraphs

To add paragraphs we use the <p><p> tag.

Exercise 2

Adding A Table

To add a table we start by using <table>
Every row should be wrapped by <tr> </tr>
Every cell should be wrapped by <td> </td>
We can wrap the Head with <thead> </thead>
We can wrap the Body with <tbody> </tbody>
We can wrap the Footer with <tfoot> </tfoot>
We end by using </table>
